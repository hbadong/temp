# 多租户架构

## 概述

AI员工系统采用多租户 SaaS 架构，所有租户共享同一套代码和数据库实例，通过 `tenant_id` 实现数据逻辑隔离。

## 数据隔离

### 共享数据库，逻辑隔离

所有业务表包含以下公共字段：

| 字段 | 类型 | 说明 |
|------|------|------|
| `tenant_id` | UUID | 租户标识，所有查询必须过滤 |
| `created_at` | TIMESTAMPTZ | 创建时间 |
| `updated_at` | TIMESTAMPTZ | 更新时间 |
| `is_deleted` | BOOLEAN | 软删除标志 |

### BaseModel 定义

```python
class BaseModel(Base):
    __abstract__ = True
    id: Mapped[uuid.UUID] = mapped_column(UUID(as_uuid=True), primary_key=True)
    tenant_id: Mapped[uuid.UUID | None] = mapped_column(index=True)
    created_at: Mapped[datetime] = mapped_column(server_default=func.now())
    updated_at: Mapped[datetime] = mapped_column(onupdate=func.now())
    is_deleted: Mapped[bool] = mapped_column(default=False)
```

## 租户模型

```python
class Tenant(BaseModel):
    __tablename__ = "tenants"
    name: Mapped[str] = mapped_column(String(255), unique=True)
    industry: Mapped[str | None]
    status: Mapped[str] = mapped_column(default="trial")
    quota: Mapped[dict] = mapped_column(JSONB, default=dict)
    usage: Mapped[dict] = mapped_column(JSONB, default=dict)
    trial_ends_at: Mapped[datetime | None]
    expires_at: Mapped[datetime | None]
```

## 配额管理

每个租户有独立配额限制：

```json
{
  "tasks_per_month": 1000,
  "storage_gb": 10,
  "agents": 3
}
```

用量统计：

```json
{
  "tasks_used": 150,
  "storage_used": 2.5,
  "agents_used": 1
}
```

### 角色权限

| 角色 | 权限 |
|------|------|
| admin | 全部权限 |
| manager | 内容、智能体、工作流管理 |
| operator | 内容创建、线索跟进 |
| viewer | 只读 |

## 租户生命周期

```
注册 → 试用(trial) → 激活(active) → 过期(expired) → 暂停(suspended)
```

- **试用**: 注册后自动进入，有期限限制
- **激活**: 付费订阅后
- **过期**: 订阅到期未续费
- **暂停**: 违规或主动暂停
